 Project Overview
This project predicts whether a customer is likely to be a credit risk using Machine Learning.

1) Features
- Credit Risk Prediction
- Data Analysis and Visualization
- Feature Importance Analysis
- Model Training and Evaluation
- Interactive Application using Flask

2) Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Excel

3) Files 
- app.py
- train_model.py
- generate_dataset.py
- credit_report.pdf
- credit_risk_model.pkl

4) How to Run
1. Install dependencies
pip install -r requirements.txt

2. Run the application
streamlit run app.py
