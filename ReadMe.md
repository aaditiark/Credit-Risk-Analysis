 Project Overview
This project predicts whether a customer is likely to be a credit risk using Machine Learning.

1) Features
- Credit Risk Prediction
- Data Analysis and Visualization
- Feature Importance Analysis
- Model Training and Evaluation
- Interactive Application using Flask

2) Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Flask
- Matplotlib
- Excel

3) Files Included
- app.py : Flask Application
- train_model.py : Model Training Script
- generate_dataset.py : Dataset Generation
- credit_risk_model.pkl : Trained Model
- scaler.pkl : Data Scaler
- credit_report.pdf : Project Report

4) How to Run
1. Clone the repository
git clone https://github.com/aaditiark/Credit-Risk-Analysis.git

2. Install dependencies
pip install -r requirements.txt

3. Run the application
python app.py
